# Frontend

Vite + React + TypeScript UI для разбора предложений.

## Запуск

```bash
cd /home/ivan/Study/dicklom/proto_py/front
npm install
npm run dev
```

## Структура

- `src/api/` — HTTP-клиент и типы API;
- `src/hooks/` — состояние сценариев;
- `src/components/` — UI-компоненты;
- `src/App.tsx` — композиция экрана.
