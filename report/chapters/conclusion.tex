\ssr{ЗАКЛЮЧЕНИЕ}

    Поставленная цель была достигнута: разработана база данных для хранения и обработки оценок трудовой эффективности сотрудников группы компаний. 
    
	В ходе выполнения работы были выполнены следующие задачи:
\begin{itemize}
	\item проведен анализ предметной области, выделины основные сущности и связи между ними, сформулированы требования и ограничения к разрабатываемой базе данных;
	\item спроектированы сущности базы данных и определины ограничения целостности данных, спроектирована ролевую модель на уровне базы данных;
	\item выбраны средства реализации базы данных и приложения;
	\item реализованы сущности базы данных и ограничения целостности. Описаны методы тестирования и разработаны тестовые случаи для проверки корректности работы функционала;
	\item проведено исследование производительности базы данных при увеличении объема данных и количестве одновременно выполняемых запросов, а также с использованием кеширования и без него.
\end{itemize}
