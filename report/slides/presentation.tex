	\begin{frame}{\centering Выводы}
		\begin{columns}[T]
			\tiny
			\begin{column}{0.5\textwidth}
				Исходя из результатов сравнения методов разрешения анафоры были сделаны следующие выводы:
				\begin{itemize}
					\item[---] большинство рассмотренных методов разрешения анафоры требуют морфологических и синтаксических характеристик;
					\item[---] большинство рассмотренных методов сосредоточены на разрешении местоименной анафоры;
					\item[---] среди рассмотренных нет единого метода для разрешения местоименной и кореферентной анафоры;
					\item[---] методы, основанные на машинном обучении, могут быть применимы к различным языкам при наличии размеченных данных для обучения;
					\item[---] в разрешении местоименной анафоры методы основанные на машинном обучении показывают большую точность, около 65\%.
				\end{itemize}
			\end{column}
			\begin{column}{0.5\textwidth}
				Исходя из результатов сравнения методов морфологического анализа были сделаны следующие выводы:
				\begin{itemize}
					\item[---] методам, не основанным на машинном обучении, требуется грамматических словарь для определения морфологических характеристик;
					\item[---] методам, не основанным на машинном обучении не требуется контекст слоформы для определения морфологических характеристик, однако в случае омонимии, такие методы не дают однозначного разбора;
					\item[---] методы основанные на конечных автоматах (FST) не могут быть применены к различным языкам;
					\item[---] большинство рассмотренных методов способны обрабатывать не словарные словоформы;
					\item[---] алгоритмы стемминга показывают значения скорости обработки в десятки тысяч словоформ в секунду;
					\item[---] точность (accuracy) рассмотренных методов превышает 90\%;
					\item[---] подход с использованием Meta-BiLSTM показывает наибольшую точность (accuracy) в получении морфологических характеристик среди методов допускающих использование в различных языках.
				\end{itemize}
			\end{column}
		\end{columns}
	\end{frame}

	\begin{frame}{\centering Выводы}
		\tiny
		Исходя из результатов сравнения методов синтаксического анализа были сделаны следующие выводы:
		\begin{itemize}
			\item[---] большинство рассмотренных методов построения деревьев зависимостей имеют линейную вычислительную сложность;
			\item[---] большинству рассмотренных методов построения синтаксической разметки требуются морфологические характеристики словоформ;
			\item[---] фундаментальным ограничением большинства рассмотренных методов является требование проективности языка;
			\item[---] большинство рассмотренных методов могут быть применены к различным языкам.
		\end{itemize}
		
		Исходя из результатов сравнения существующих систем управления корпусными данными сделаны следующие выводы:
		\begin{itemize}
			\item[---] наибольшую степень автоматизации выполнения разметок и выравнивания предоставляют системы управления параллельным корпусом НКРЯ и татарско-русским корпусом <<Туган тел>>;
			\item[---] ни одна из систем управления корпусными данными не обеспечивает полностью автоматизированного выполнения разметок и выравнивания.
		\end{itemize}
	\end{frame}
